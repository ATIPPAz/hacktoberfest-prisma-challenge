## Setup

1. install dependecies

   **pnpm**
   `pnpm install`

   **npm**
   `npm install`

   **yarn**
   `yarn install`
   <br/>

2. run migration files

   **pnpm**
   `pnpm run migrate`

   **npm**
   `npm run migrate`

   **yarn**
   `yarn run migrate`
   <br/>

3. run seeding data to SQLite database

   **pnpm**
   `pnpm run seed`

   **npm**
   `npm run seed`

   **yarn**
   `yarn run seed`
   <br/>

4. run prisma studio to check

   **pnpm**
   `pnpm run studio`

   **npm**
   `npm run studio`

   **yarn**
   `yarn run studio`

   <br/>
